# Project Title
MassageRx
## Overview
MASSAGERX IS A PYTHON BASED PROGRAM THAT USES USER RESPONSES TO PROVIDE SUGGESTION ON WHAT MASSAGE THEY SHOULD GET. 
THIS PROGRAM IS TO BE USED A MARKETING/NEW CLIENT CAPTURE TOOL TO DECREASE THE DECISION MAKING BURDEN FOR THE CLIENT. 
MINIMIZE RISK OF BOOKING THE WRONG MASSAGE TYPE, DURATION AND ADD ON. PRACTICE INCREASES CLIENT CAPTURE AND REVENUE FROM ADD-ONS.
## Why I Built This
THIS PROGRAM IS SPECIFICALLY FOR BUSINESS PURPOSES. THOUGH HIGH LEVEL RESEARCH IS LIMITED, THERE IS SOME RESEARCH THAT SUGGESTS CERTAIN TYPES
OF MASSAGE ARE BETTER FOR CERTAIN BODY PARTS, CONDITIONS AND PREPARATION FOR EVENTS. (?) THERE IS PROBABLY RESEARCH THAT SHOWS A LARGES STRUGGLE
FOR MASSAGE THERAPY PRACTICES IS MARKETING, SETTING THEMSELVES APART AND CAPTURING NEW CLIENTS
## Features
RIGHT NOW THE PROGRAM IS BUILT IN PYTHON. USES FLASK FOR END USER. COLLECTS USER INPUT. STORES DATA WITH MYSQL. SUGGESTS TYPE, DURATION AND ADD ONS
FOR MASSAGE APPOINTMENT. PROVIDES COST ESTIMATE FOR MASSAGE APPOINTMENT BASED ON SUGGESTION.
